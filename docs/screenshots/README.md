# Screenshots Directory

This directory contains screenshots of the CAPTCHA interface.

## How to Generate Screenshots

1. Start the server:
   ```bash
   python app.py
   ```

2. Open in browser:
   ```
   http://localhost:5000
   ```

3. Capture different states:
   - Main challenge screen
   - Success state (correct answer)
   - Failure state (incorrect answer or bot detected)
   - Different challenge types

## Required Screenshots

- `captcha-interface.png` - Main interface
- `challenge-screen.png` - Challenge display
- `success-state.png` - Success message
- `failure-state.png` - Error message
- `logic-puzzle.png` - Logic puzzle challenge
- `math-problem.png` - Math problem challenge
- `pattern-recognition.png` - Pattern challenge
- `contextual-question.png` - Contextual question
- `behavioral-dashboard.png` - Admin view (optional)
